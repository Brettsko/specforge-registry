# specforge-registry

Verified Golden Specs for [specforge](https://github.com/Brettsko/specforge) — battle-tested blueprints that go straight to your coding agent without touching the council.

## Available specs

| Name | Description | Verified |
|------|-------------|---------|
| `jwt-auth-server` | JWT auth with register, login, refresh tokens, and protected routes. bcrypt + SQLite. | ✅ 10/10 |
| `stripe-subscriptions` | Stripe webhook handling, customer creation, subscription tracking. SQLite. | ✅ 7/7 |

## Usage

```bash
# See what's available
specforge list

# Fetch a spec
specforge pull jwt-auth-server --save

# Build it — no council, straight to code
specforge run --from-spec .specforge/specs/jwt-auth-server.md --project-dir ./my-auth
```

## What is a Golden Spec?

Every spec in this registry was generated by specforge's council of AI agents, built by a coding agent, and verified by an automated test script. Nothing is hand-authored. If it's in this registry, it passed.

## Adding specs

Specs are added when a specforge run passes verification on the first or second round. The verify.sh script is included alongside every spec.md so you can re-run verification yourself.
